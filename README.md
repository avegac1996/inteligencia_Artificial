# Taller de Python y NumPy

Notebooks del taller de Python y NumPy para el curso de Inteligencia Artificial.

## Contenido


**Modulo 1:
- Multiplos de 7 con list comprehension
- Filtrado de diccionarios
- Funcion resumen con estadisticas
- Map con funcion de potencia

**Modulo 2:
- Creacion de vectores, matrices y puntos con linspace
- Atributos de arrays (shape, dtype, nbytes)
- Diferencia entre enteros y flotantes en NumPy
- Matriz identidad y traza

**Modulo 3:
- Extraccion de submatrices, columnas y esquinas
- Mascaras booleanas y reemplazo condicional
- Reshape a diferentes dimensiones (2D, 3D, vector fila)
- Demostracion de vista vs copia

**Modulo 4:
- Estadisticas por columna y normalizacion
- Broadcasting con vectores fila y columna
- Argsort para ordenar arrays paralelos
- Concatenacion vertical/horizontal y division
